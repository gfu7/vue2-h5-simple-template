# vue2-h5-simple-template

## 项目框架信息：
主要面向移动端的基本项目结构。  
渲染框架：vue2（比vue3更高的兼容性）  
UI框架：Vant(vue2版本)  
CSS预处理：less  
脚手架：vue-cli(v4.x) / webpack4  
代码lint：默认airbnb  
响应式适配方案：rem  
国际化方案：vue-i18n(v8.x)




## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
